# 地質図のはなし（中学生~高校生向け）

2019年1月12日

* 本テキストは[筑波大学地層学研究室](http://www.geol.tsukuba.ac.jp/~strati/)と[株式会社ダンク](https://www.nsb-dank.co.jp)によって共同作成されました。

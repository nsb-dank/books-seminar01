# 1-3. 地質図のつくり方

地質図は、下図で示したような流れで作成しますが、基本は以下の手順になります。  
1. **地表踏査を行う**  
2. **ルートマップを作る**  
3. **地質図を作る**

地表踏査で得られた観察結果やサンプルが直接証拠になり、それに地質図学や様々な状況証拠を加えて作成していきます。
